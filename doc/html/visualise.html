visualise -- plots the appearance of Roche-distorted stars in a binary

visualise computes grids of points representing two stars and optionally a disc and plots those visible at a particular phase projected onto the plane of the sky. It can only handle stars less than or equal to their Roche lobes. This allows visualisation and checking of the eclipse computations.

Command invocation

visualise model nphase nphase (phase)/(phase1 phase2) device x1 x2 y1 y2

Arguments

model---Parameter file defining the model, as used e.g. by lrcohe.
nphase---Number of orbital phases to display.
phase---Orbital phase if nphase=1
phase1---First orbital phase if nphase>1
phase2---Last orbital phase if nphase>1
device---Plot device
x1---left-hand X limit
x2---right-hand X limit
y1---lower Y limit
y2---upper Y limit
width---width of plot in inches
reverse---reverse black and white (or not)

Classes

This command is a member of the class: Model.
