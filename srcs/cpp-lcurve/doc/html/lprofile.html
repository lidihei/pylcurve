Computes the line profile from a Roche distorted star

lprofile computes the line profile from a Roche distorted star, allowing for irradiation of one face. The radiation from the star is modelled as a slab of constant optical depth with a source function that changes exponentially with depth. This allows one to have a continuous transition from optically thin to thick and have limb darkening or brightening. The computation is carried out for the secondary star only.

model---File of parameters specifying the parameters. See lroche for a full description of these.
tau---Total vertical optical depth of emitting layer. Allows one to simulate optically thin (tau << 1) and optically thick (tau >> 1) cases.
efac---Source function changes as exp(efac*tau). This must be less than 1. efac > 0 is equivalent to the usual limb darkened case. efac < 0 implies limb brightening.
aline---Parameter controlling strength of absorption line which will have limb and gravity darkening as defined by the model file, i.e. it is assumed to be of constant equivalent width, which may not be accurate of course.
nphase---number of phases
phase---Phase to compute (if nphase == 1)
phase1---First phase to compute (nphase > 1)
phase2---Last phase to compute (nphase > 1)
store---molly file to store results (nphase > 1)
expose---Length of exposure in terms of orbital phase
ndiv---Number of sub-divisions to account for finite exposures. The results will be trapezoidally integrated over the length of the exposure.
v1---left hand edge of profile array (same units as vscale)
v2---right hand edge of profile array (same units as vscale)
nbin---number of velocity bins
fwhm---full-width half maximum of any blurring (same units as vscale)
nfine---fine sub-division factor for blurring. Each contribution is added into the nearest pixel in a fine array and then blurred.
align---true to shift out the motion of the centre of mass of the secondary star. This allows the profiles to be compared more easily although not necessarily directly with data.

Classes
