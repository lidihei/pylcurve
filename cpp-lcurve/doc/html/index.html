lcurve commands

The following comands are available:

levmarqfits light-curve to a binary star model using Levenberg-Marquardt
lprofilecomputes the line profile from irradiated face of a Roche distorted star
lrochecomputes the light-curve of a sphere and a Roche distorted star
picturedraws a Roche lobe, cvmovie style
simplexfits light-curve to a model with a sphere plus Roche-lobe distorted star using simplex algorithm
visualiseplots the appearance of Roche-distorted stars in a binary
